GLD Operator — console revision

8 layout candidates, live-tested

Console yang sekarang menampilkan semua modul terbuka sekaligus, jadi satu layar bisa penuh sebelum operator butuh separuhnya. Delapan pola di bawah menyembunyikan detail sampai diminta — accordion, drawer, popover, wizard, sheet — sambil tetap satu klik dari command aslinya. Semua contoh interaktif langsung, coba klik.

01

Collapsible module accordion

Running tab

Ganti "Power and Boot Health details" yang sudah pakai <details> jadi pola accordion konsisten untuk semua modul sekunder (Power, Boot Health, Sensor Check meta). Default collapsed, state per-modul diingat.

Wizard-step popup

Dataset start

Dataset tab sekarang satu form panjang (parameter + WiFi/MQTT sekaligus). Versi wizard memecah jadi 3 langkah kecil dalam satu popup, operator tidak scroll form panjang dan urutan pengisian jelas.

07

Unified danger-action popup

Firmware / Inject ID

Upload Firmware, Inject ID, dan Apply GLD Settings sekarang tiga confirm() browser generik yang tampilannya beda dari aplikasi. Satu komponen popup yang menampilkan command persis yang akan dikirim, konsisten untuk ketiganya.

Confirm firmware upload

This flashes COM10 and will disconnect serial during upload.

pio run -e gld -t upload --upload-port COM10
08

Fleet quick-switch popover

Header badge

Ganti slot aktif sekarang lewat sidebar Fleet, yang berarti scroll ke bawah dulu kalau lagi di tab lain. Klik badge device di header langsung membuka daftar 8 slot untuk pindah tanpa keluar dari tab yang sedang dibuka.

Cara pilih: sebut nomornya saja (misal "pakai 1, 3, dan 7") — masing-masing pola independen, jadi bisa dikombinasikan tanpa bentrok. Semua tetap dalam palet dan font yang sama dengan console yang sudah ada, jadi tidak perlu redesign ulang seluruh app.
