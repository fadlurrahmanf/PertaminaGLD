LPG Consistency: Rank Order Shifts with Concentration

Finding: The sensor rank order is NOT consistent across LPG trials. At low concentration, MQ2 dominates. At high concentration, MQ135 overtakes MQ2. This means a single threshold cannot classify LPG — the model must learn concentration-dependent patterns.

Key Insights

1. MQ sensitivity order is NOT consistent across LPG concentrations. Low conc: MQ2 > MQ6 > MQ5. High conc: MQ135 > MQ6 > MQ7. The dominant sensor shifts. ML model must learn this.
2. Within same concentration, rank order IS consistent. All 9 low-conc events in multi-file show MQ2 as #1. All high-conc events show MQ135/MQ7 as #1. Repeatability is good within concentration band.
3. MQ6 is the most consistently strong sensor. Always in top 3-4 across all gases and concentrations. Most reliable cross-gas indicator.
4. O2 has only 1 run. Cannot assess consistency. MQ6 dominant (0.058V) but signal is 43x weaker than LPG. Need more O2 trials.
5. CO2 has only 1 run. MQ135 dominant (0.313V) with clean response. Need more CO2 trials to confirm consistency.
6. MQ8 and MQ3 are blind. Perfectly consistent near-zero response across ALL gases. These are reliable negative controls.
