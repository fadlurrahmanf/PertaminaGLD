# ActivityAI

Folder koordinasi kerja AI-agent di repo ini.

| Path | Fungsi |
|---|---|
| `codexactivity.md` | Log aktivitas Codex |
| `claudeactivity.md` | Log aktivitas Claude |
| `rules/` | Aturan kerja Codex/Claude dan workflow paralel |

Sebelum AI-agent melakukan kerja non-trivial, agent wajib membaca activity log
agent lain dan menulis rencana di log miliknya sendiri.
