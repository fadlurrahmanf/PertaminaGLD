Implications for Multi-Gas Classification

These consistency results directly inform the ML classification strategy.

  1. Clean air baseline is consistent — sub-mV readings across all sensors with minimal drift, providing a stable reference for event detection.
  2. Each gas has a distinct 8-sensor fingerprint — the normalized response pattern uniquely identifies gas type, not just a single sensor threshold.
  3. CO2 is the easiest to classify — 100% MQ135 dominance across all 4 events. MQ135 always >10x any other sensor. Simple threshold works.
  4. LPG requires concentration-aware classification — at low dose MQ2 dominates; at medium/high dose MQ135 dominates. The top-4 set (MQ135, MQ2, MQ6, MQ7) is consistent but ranking shifts with concentration.
  5. O2 needs more data — only 1 significant event detected. MQ6 is clearly dominant (1.0 vs 0.156 for next). More injection cycles needed for confidence.
  6. ML model should learn fingerprint patterns — a random forest or neural network operating on the 8-dimensional normalized response vector will capture both gas identity and concentration effects.
