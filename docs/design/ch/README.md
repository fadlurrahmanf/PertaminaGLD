# CH Design

This folder stores the Cluster Head design baseline for the Pertamina GLD monorepo.

- `design.md` is imported from `D:\ClusterHeadDesign\Design CH_24052026.md`.
- Treat `design.md` as the immutable imported baseline.
- Use `final_design.md` as the current CH implementation reference generated from the live firmware source.
- `design.updated.draft.md` is historical draft material and is not the current implementation contract.
- Use `docs/design/gld-ch/payload-contract.draft.md` for the current GLD-CH payload and crypto contract.
- If the source document changes outside this repo, re-import it here and record the reason in the project decision log.
